            기본 흐름 보기
            

              
왼쪽 요소 목록에서 텍스트, 이미지, 버튼, 카드 같은 요소를 눌러 캔버스에 추가합니다.

              
캔버스에서 요소를 드래그해 위치를 옮기고, 오른쪽 아래 조절점으로 크기를 바꿉니다.

              
오른쪽 속성 패널에서 텍스트, 색상, 모양, 클릭 동작, 화면 번호를 편집합니다.

              
미리보기에서 버튼 동작과 화면 전환을 확인하고, 코드 생성에서 HTML/CSS/JavaScript를 복사합니다.
